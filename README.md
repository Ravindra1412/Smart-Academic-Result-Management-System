# Smart Academic Result Management System

## 📌 Description
A web-based system to manage and analyze student academic results efficiently.

## 🚀 Features
- Add student details
- Enter and update marks
- Calculate total and percentage
- Generate result reports

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download or clone the project
2. Open `Final2.html` in browser
